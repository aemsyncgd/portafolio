# Portafolio — Arturo Molleda

Portafolio profesional construido con [Astro](https://astro.build) 6 + [Tailwind CSS](https://tailwindcss.com) 4, basado en el tema [NULLTRACE](https://github.com/andreialba/nulltrace).

Estética terminal oscura, adaptable, con animaciones tipo typewriter y conteo de impacto al hacer scroll.

## Secciones

- **Especializaciones** — Desarrollo Web Full-Stack, Automatización & DevOps, IA Local, Sistemas de Gestión
- **Skills** — Tecnologías y lenguajes
- **Stack** — Inventario técnico con categorías (frontend, backend, devops, ai-local)
- **Proyectos** — 9 proyectos reales desde GitHub
- **Logros e Hitos** — Trayectoria y metas alcanzadas
- **Contacto** — Formulario con envío por email

## Desarrollo

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # Salida estática en dist/
npm run lint     # TypeScript check
```

## Deploy

El sitio genera HTML estático. Compatible con Vercel, Cloudflare Pages, Netlify o cualquier hosting estático.
